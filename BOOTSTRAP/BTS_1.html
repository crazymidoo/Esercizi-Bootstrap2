Cos'è Bootstrap?

Nato nel 2010 come un progetto interno a Twitter a opera di Mark Otto e Jacob Thornton, Bootstrap è una collezione open source di strumenti grafici, stilistici e di impaginazione che possono essere utilizzati per agevolare lo sviluppo di applicazioni web responsive e mobile-first.


Cosa devi importare per creare una pagina web con bootstrap?

Per creare una pagina web utilizzando Bootstrap ci sono due modi:

  • Scarico il file CSS dal sito ufficiale di Bootstrap;
  • Importo il file CSS da remoto con il seguente link: <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
